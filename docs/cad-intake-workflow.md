# CAD Intake Workflow

Intended workflow:

1. CAD received
2. File validation
3. CAD layer inventory
4. Coordinate system review
5. Import into staging
6. Source-to-target layer mapping
7. Geometry conversion
8. Attribute normalization
9. QA checks
10. Comparison with existing assets
11. Reviewer approval
12. Production update package

Production loading is not implemented.
