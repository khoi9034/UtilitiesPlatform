All mappings default to `approved_to_standardize = false`. Units, code meanings, authoritative IDs, source owners, and elevation datum remain unconfirmed until a data owner approves them.

Curated data is not created yet because review decisions, owner confirmation, dependency gaps, and standardization approval are still pending.
