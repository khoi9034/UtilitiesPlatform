# Adding A New Dataset

1. Confirm authorization to use the data.
2. Copy or export the approved source into the correct raw folder.
3. Preserve the original file unchanged.
4. Register it in the catalog.
5. Inventory its layers and fields.
6. Import selected layers into staging.
7. Run QA checks.
8. Standardize schema and coordinate system.
9. Approve selected data for curated storage.
10. Create controlled exports only when authorized.

Raw source files are the local record of what was received. Do not edit them in place. Use staging and standardized folders for working copies.
