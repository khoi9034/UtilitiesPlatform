# GIS Workspace

This folder is for safe GIS processing utilities, schema references, CAD intake helpers, QA scripts, notebooks, and toolbox assets.

ArcPy scripts must be run inside an ArcGIS Pro Python environment. The placeholder modules in this repository defer ArcPy imports and return safe `not_configured` responses until real workspace paths, schemas, and review rules are supplied.

No county paths, credentials, or destructive editing operations belong in these scripts.
